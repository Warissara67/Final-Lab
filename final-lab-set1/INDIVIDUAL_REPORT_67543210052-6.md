## ปัญหาที่พบและวิธีการแก้ไข

### 1. Frontend ไม่สามารถเรียก API ได้

สาเหตุ:
เรียก API ไปที่ localhost โดยตรง ทำให้ไม่ผ่าน Nginx Gateway

วิธีแก้ไข:
เปลี่ยน endpoint เป็น /api/... เพื่อให้ request วิ่งผ่าน Nginx


---

### 2. CORS Error จาก Browser

สาเหตุ:
Frontend และ Backend อยู่คนละ origin

วิธีแก้ไข:
ตั้งค่า Nginx ให้เป็นตัวกลาง (API Gateway) เพื่อลดปัญหา CORS
---

### 3. หน้าเว็บไม่แสดงผลหลังจากรัน Docker

สาเหตุ: 
เข้าไฟล์ HTML โดยตรง (file://) แทนการเข้า server

วิธีแก้ไข:  
ข้าผ่าน http://localhost:8080 แทน

---

### 4. UI ไม่อัปเดตหลังแก้ไขโค้ด

สาเหตุ:  
Browser cache เก็บไฟล์เก่าไว้

วิธีแก้ไข:  
ล้าง cache หรือใช้ Hard Reload
